# Hong Kong Boundary Resources

`hk_18_districts.geojson` is a local copy of the Hong Kong administrative district boundary GeoJSON.

- Source: Home Affairs Department via DATA.GOV.HK
- Download URL: https://www.had.gov.hk/psi/hong-kong-administrative-boundaries/hksar_18_district_boundary.json
- Retrieved: 2026-06-21
- Feature count: 18
- Canonical district-name property: `District`

This resource replaces `FAO/GAUL/2015/level2` for the Hong Kong smoke and monthly NDVI workflows because live probing showed GAUL level 2 does not expose Hong Kong district names suitable for filtering `Central and Western`.
